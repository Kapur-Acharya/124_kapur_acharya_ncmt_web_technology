## What's Included

**Syllabus & Summary**
- Exact syllabus topics for the chapter
- Chapter summary with introduction, key concepts, and learning outcomes

**JavaScript Topics**
- let / const — variable declaration and scope
- Arrow functions — syntax and this behavior
- Template literals — string interpolation and multi-line strings
- Destructuring — array and object destructuring
- map(), filter(), reduce() — array transformation methods
- Spread / rest operators
- DOM manipulation — selecting elements, events, event delegation
- Need for interactive web pages & replacing Java Applets with JS
- Browser storage — localStorage, sessionStorage, cookies

Each .js file includes a multiline comment explaining the topic, its purpose, syntax, and a working code example.

## How to Run
Open any .js file in a code editor or run it using Node.js / browser console to see the examples in action. .txt files can be opened in any text editor.
